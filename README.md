# Attenda - SaaS CRM para Eventos y Gestión de Invitados

Attenda es una plataforma premium diseñada para simplificar la organización de eventos complejos, la gestión de listas de invitados y el seguimiento de tareas en tiempo real.

## 🚀 Estado del Proyecto
Actualmente, la plataforma cuenta con:
- **Backend robusto** en .NET con Clean Architecture.
- **Frontend premium** en React con Dashboard interactivo.
- **Integración total** con Supabase para Auth y Edge Functions.
