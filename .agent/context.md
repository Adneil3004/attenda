# EventFlow RSVP - Documento del Proyecto (MVP)

## 1. Descripción del Proyecto
**Attenda** es un SaaS tipo CRM diseñado para la gestión integral de invitados en eventos sociales y corporativos (bodas, cumpleaños, reuniones, etc.). La plataforma centraliza la logística del evento, desde el primer contacto hasta el acceso el día del evento.

### Características Principales (MVP):
- **Confirmación RSVP Inteligente:** Enlaces únicos por invitado para confirmación de asistencia en tiempo real.
- **Gestión de Invitados:** Dashboard para organizar listas, dietas especiales y grupos.
- **Coordinación de Tareas:** Herramientas para que los organizadores gestionen pendientes del evento.
- **Check-in Digital:** Generación de boletos digitales con códigos QR para un acceso fluido.

## 2. Objetivo
Ofrecer una solución simple, elegante y eficiente para organizar eventos, gestionar invitados, confirmar asistencia y coordinar tareas en un solo lugar.
