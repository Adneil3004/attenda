# Concierge Design System (Aesthetics) - Attenda

The "Concierge" design system is the soul of the Attenda user interface. It is characterized by its elegance, clarity, and premium feel, designed to make event organizers feel in control and sophisticated.

## 💎 Visual Core Principles

### 1. Glassmorphism & Depth
Every management panel (modals, drawers, and the sidebar) should feel like a layer on top of the event's data.
- **Recipe**: Translucent background + Backdrop Blur + Subtle Border.
- **Token Usage**: `bg-[var(--color-primary)]/10 dark:bg-black/40 backdrop-blur-sm`.
- **Contrast**: Border color should be `--color-surface-container-low` with low opacity.

### 2. Surface Hierarchy
We use a token-based surface system to manage depth in both Light and Dark themes.
- **Level 0 (Lowest)**: Used for main dashboard panels and cards. (`--color-surface-container-lowest`)
- **Level 1 (Low)**: Used for hover states and secondary elements. (`--color-surface-container-low`)
- **Level 2 (Normal)**: Used for the main page background. (`--color-surface-container`)

### 3. Typography
- **Headings**: Use **Outfit** or **Inter** with medium to semibold weights for clear hierarchy.
- **Body**: Use **Inter** for legibility. Avoid overcrowding text.

## 🌙 Dark Mode Standards

Dark mode isn't just "inverted colors"; it's a shifted atmosphere.
- **Badges**: Use semi-transparent backgrounds (`/10` or `/20` opacity) with full-intensity text colors. (e.g., `bg-red-500/10 text-red-500`).
- **Inputs**: Inputs should use the `lowest` surface color and have subtle borders. Focus states use the primary brand tint.
- **Transitions**: Every theme swap should be instantaneous but smooth, ensuring no "flash" of unstyled white content.

## ✨ Patterns & Polish

### The "Sheen" Effect
Interactive elements should occasionally feature a "sheen" or "shimmer" animation to guide the user's eye and reinforce the premium identity.
- **Utility**: `animate-shimmer` on a relative container with an absolute pseudo-element.

### Micro-animations
- **Modals**: Always use a subtle scale and opacity entry (`scale-95` to `scale-100`).
- **Lists**: Items should transition their background colors over 200ms on hover.
